Copa do Mundo 2026 - Banco de Dados
Descrição

Projeto desenvolvido para a disciplina de Banco de Dados com o objetivo de modelar e implementar um sistema para gerenciamento das informações da Copa do Mundo 2026.

O banco de dados permite armazenar:

Países participantes
Seleções (times)
Jogadores
Jogos
Resultados das partidas
Escalações
Grupos da competição
Estrutura do Banco

O projeto possui as seguintes tabelas:

Pais

Armazena os países participantes da competição.

Grupo

Armazena os grupos da Copa do Mundo.

Time

Armazena as seleções participantes.

Jogador

Armazena os jogadores de cada seleção.

Jogo

Armazena as partidas da competição.

Resultado

Armazena os resultados dos jogos.

Escalacao

Armazena os jogadores escalados em cada partida.
